Flash Gestures

Use your favorite mouse gesture add-on and Firefox hotkeys on plugins like Flash Player

Download as .zip Download as .tar.gz View on GitHub

A firefox add-on that resolves the mouse gestures/hotkeys usability problem when plugins (like Adobe Flash Player) gain focus.

Install

Flash Gestures is a bootstrapped (no restart) Firefox add-on. Download the add-on file FlashGestures-1.0b-unified.xpi and open with Firefox to install it.
